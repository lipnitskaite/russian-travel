## Project Description

It is a project about travelling across Russia for the Yandex Practicum Web development course. <br>

It allowed me to practice CSS Grid Layout and CSS Flexbox Layout for elements positioning. I also used the media queries to adapt the landing to 320, 768, 1024, and 1280px devices.<br>
